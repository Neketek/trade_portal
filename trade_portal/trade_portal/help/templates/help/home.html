Trade Portal Documentation


The trade portal is used to provide UI and API access to any authorized party wanting to send the documents.

Although this party may integrate to the IGL node or OA notarisation services independently - it's impractical for a large number of traders and chambers.

The basic workflow to use the website is:

  1. Register
  2. Confirm access to some organisation (using business identifier - ABN for Australia)
  3. Start sending documents using UI
  4. or acquire API access and use API to submit them.
  5. Observe incoming documents for own parties
